## 1. Overview
FinWise AI is an intelligent loan eligibility, credit analysis, and financial advisory platform. It gives users instant, explainable insight into their loan eligibility before they formally apply — reducing uncertainty for applicants and unnecessary manual screening for financial institutions.

## 2. Problem Addressed
Lack of early eligibility awareness: users apply for loans without knowing if they'll qualify, leading to rejections, wasted time, and credit-report damage.

## 3. Key Features
- Instant loan eligibility analysis
- Credit score evaluation
- EMI calculator
- Personalized financial recommendations (Groq AI-generated)
- Applicant history dashboard

## 4. Social Impact
Promotes financial awareness and responsible borrowing by helping users understand their financial profile before making borrowing decisions — leading to better debt control and credit behavior over time.

## 5. Business Model / Impact
Can be offered to banks, financial institutions, and fintech companies as a **customer engagement and pre-screening tool** — reducing manual workload, filtering likely-eligible applicants, and improving the overall customer journey.

## 6. Existing Solution Benchmark
Paisabazaar Loan Eligibility Calculator — static and formula-based. FinWise AI differentiates through AI-driven, personalized, explainable guidance instead of a binary result.

## 7. Reference
TransUnion (credit scoring and evaluation standards)

## 8. Approach
Design-thinking-driven development: Empathize → Define → Ideate → Prototype → Test, followed by Agile sprints for build-out.

## 9. Requirements (Hardware/Software)
- Development laptop with Python 3.x installed
- Groq API key (free tier sufficient for demo)
- Internet connection for AI API calls
- Browser for frontend testing

## 10. Recommended Technology Stack
Python, Flask, Groq AI, SQLite, Ngrok
